Mobiscroll Select Scroller

by Acid Media

Renders a scrollable list from HTML selects for easy data picking on touch devices. It comes with theming capability, optional jQuery Mobile integration and multiple display modes: Inline, Modal, Bubble, Docked Top & Bottom. It works with optgroups as well.
